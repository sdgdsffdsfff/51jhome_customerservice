注意：

1、【更新】【2014-07-09】GROUP_TPL_DIR 可自己在每个分组父级控制器初始化时自行设置，如果设置了GROUP_TPL_DIR，则当前分组模板目录以指定的目录为准，否则将默认采用当前分组名下的模板;

例如：当前分组为“admin”，则默认模板路径为：tpl/[模板样式]/admin/[控制器]/[模块]_tpl.php，但是如果在调用模板之前设置了常量GROUP_TPL_DIR，则变成：tpl/[模板样式]/[GROUP_TPL_DIR]/[控制器]/[模块]_tpl.php，利用好GROUP_TPL_DIR，可以让模板调用更灵活。

class commonAction extends action {
      //初始化全局信息
      function __construct() {
           //设置后台模板目录
           define('GROUP_TPL_DIR', 'content');
      }
}

 

2、TPL_PATH、PUBLIC_PATH、IMG_PATH、JS_PATH、CSS_PATH只会在调用模板输出数据时才定义，因此只可在模板中使用;

3、WEB_URL与SITE_PATH的区别，当在config/config_inc.php中定义了main_path时，SITE_PATH值为main_path的值 ;
